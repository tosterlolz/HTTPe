HTTPE Server

This server accepts PGP-encrypted POST requests (Content-Type: application/httpe+pgp) and replies with an encrypted, signed response.

Status

Server is running. Use the CLI client, browser extension or custom client to POST encrypted data.

Keys

Place keys in the default locations (e.g. ./.well-known/httpe/ or configure with the KEYS_DIR environment variable).

Example

POST / HTTP/1.1
Host: example.local:533
Content-Type: application/httpe+pgp

<PGP MESSAGE>

Check server log for decryption/signature verification output.
